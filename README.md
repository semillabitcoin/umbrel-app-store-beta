# Semilla Bitcoin — Community App Store (Beta)

Canal **beta** para probar versiones candidatas de las apps de Semilla Bitcoin
en Umbrel antes de publicarlas en la tienda estable
(`semillabitcoin/umbrel-app-store`).

Es opt-in por dispositivo: se añade por URL en Umbrel → App Store → Community
App Stores. Ahora mismo **sin apps publicadas**.
